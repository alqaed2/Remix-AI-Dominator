# 🚀 Remix: AI DOMINATOR Engine
### نظام تشغيل النمو المتقدم والمحاكاة الجينية الذكية للمحتوى الفايرل (Creator Genome Cloud)

---

## 📌 مقدمة عن النظام
**Remix: AI DOMINATOR** هو منصة ريادية متكاملة (Full-Stack) مُصممة خصيصاً لتمكين صناع المحتوى من السيطرة الكاملة على خوارزميات منصات الفيديو القصيرة (TikTok, Instagram Reels, YouTube Shorts) عبر تحويل عملية صناعة المحتوى الإبداعي من التخمين العشوائي إلى لغة الأرقام والهندسة الجينية للمحتوى الفايرل.

يعتمد النظام على **بنية خوادم متطورة** تدمج قوة نماذج الذكاء الاصطناعي التوليدي من **Google Gemini** مع أنظمة طوابير المهام الخلفية لمعالجة وتحليل ونمذجة جينوم منشئي المحتوى (Creator DNA) وتوليد سيناريوهات سينمائية تفصيلية وبث التوقعات التفاعلية مباشرة للمستخدم خطوة بخطوة.

## 🛡️ الميزات الأساسية والوحدات التقنية (Core Modules)

### 1. مُحلل الجينوم البصري والسمعي (Creator DNA & Niche Genome)
* **تحديد الهوية الجينية (DNA Profiling):** يستخلص النظام البنية الجينية الخاصة بأداء المنشئ الإبداعي (مثل نبرة الإلقاء، الوتيرة الحركية، التوجيه البصري، ونقاط الضعف الفنية).
* **تحليل لقطات الشاشة بالرؤية الحاسوبية (Computer Vision):** معالجة فورية للقطات شاشة إحصائيات الأداء (TikTok / Instagram Screenshots) لاستخراج الأرقام بدقة متناهية (مشاهدات، معدل الحفظ، معدل الإكمال البصري) لتقييم كفاءة الحساب تلقائياً.

### 2. بوابة ريمكس المحتوى الذكي (Content Remixing Console)
* **تحويل الأفكار والـ Remixing:** إعادة صياغة وهيكلة أي فكرة أو موضوع رائج لتتوافق بشكل مطلق مع الحمض النووي الإبداعي (Creator DNA) الخاص بالمنشئ لضمان ثبات الهوية البصرية والسمعية.
* **البث الفوري الحي (SSE Streaming):** دعم كامل للبث المباشر (Server-Sent Events) لتوليد أفكار المحتوى والسيناريوهات والخطافات الإبداعية (Visual Hooks) كلمة بكلمة في الواجهة الأمامية دون فترات انتظار طويلة.

### 3. محاكي التنبؤ بمعدل البقاء والنجاح (Success Prediction Engine)
* **المحاكاة الهيكلية قبل التصوير:** يقوم المحرك بـ "تشريح" هيكل السكربت الإبداعي ومقارنته جينياً بخصائص الجمهور المستهدف ومستوى الانتباه العام.
* **لغة الأرقام الدقيقة:** توليد نسب نجاح متوقعة (Success Probability %)، ومعدلات الإكمال المتوقعة (Completion Rate)، ومقاييس التفاعل المتوقعة، مع إبراز عوامل الخطر الإيجابية والسلبية وتوليد توصيات هيكلية تكتيكية فورية للسكربت.

### 4. نظام جدولة حزم التخريج السينماتيكي (Cinematic Pack & Job Queue)
* **مجدول العمليات الخلفي (Asynchronous Worker):** للعمليات الحسابية الضخمة التي تتطلب تخليق حزم سينماتيكية متكاملة (Cinematic Packs / Warheads)، يرسل النظام المهام لقائمة انتظار مستقرة لتفادي انقطاع الاتصال.
* **شاشة مراقبة الأنوية التفاعلية (Terminal Logs View):** تمكين المستخدم من متابعة معالجة حزمة جينات المقطع بنسبة مئوية وسجل تفاعلي تفصيلي شبيه بأنظمة تشغيل الخوادم الاحترافية.

### 5. المهام القتالية اليومية (Daily Growth Missions)
* **سد ثغرات الأداء:** يقوم النظام أوتوماتيكياً بتحليل نقاط الضعف المسجلة في الـ DNA الخاص بصانع المحتوى وتوليد مهام يومية قتالية مخصصة له (مثل تحدي كتابة خطاف غير مباشر، أو كسر وتيرة الصوت) لمساعدته في السيطرة الإبداعية واختراق الخوارزمية.

## 🛠️ التقنيات المستخدمة (Technical Stack)

* **الواجهة الأمامية:** React 19, Tailwind CSS v4 (أنماط بصرية مستقبلية وخطوط مخصصة مثل *Space Grotesk* و *JetBrains Mono*)، مكتبة **Motion** للحركات الانتقالية السلسة، و **Recharts** للمخططات البيانية للنمو التاريخي.
* **الخلفية وقاعدة البيانات:** Node.js (Express.js), TypeScript (tsx), Prisma ORM وقاعدة بيانات علائقية لتسجيل جينوم وبيانات المنشئ.
* **الذكاء الاصطناعي:** SDK الرسمي المتطور من جوجل `@google/genai` لاستخدام نماذج Gemini السحابية الفائقة، مع تطبيق الاستعلام ببث البيانات SSE وتقنية الذكاء الاستخلاصي والتنبؤ بالأداء.
* **إدارة الطوابير والمهام:** BullMQ و ioredis لضمان معالجة خلفية آمنة وثابتة للمهام ذات المدد الزمنية الطويلة.

## ⚙️ التثبيت والتشغيل المحلي (Setup & Installation)

### 1. المتطلبات الأساسية
* وجود بيئة تشغيل **Node.js v18+** مثبتة محلياً.
* مفتاح وصول رسمي **GEMINI_API_KEY** (يتم تمريره عبر خادم بيئة العمل السحابي أو ملف المتغيرات المحلي).

### 2. خطوات التثبيت والتشغيل
قم بتثبيت حزم الاعتماديات المطلوبة بالكامل:
```bash
npm install
```

لبدء تشغيل خادم التطوير التفاعلي (الذي يعمل تلقائياً على المنفذ **3000** مع دعم بيئة المعالجة والـ Hot Module Replacement):
```bash
npm run dev
```

لعمل بناء برميجي مخصص لبيئات الإنتاج (Production Build):
```bash
npm run build
```

لتشغيل خادم الإنتاج السريع والمحزم بالكامل باستخدام `esbuild`:
```bash
npm run start
```

## 🎯 معايير التصميم والجودة المتبعة (Design & Quality Standards)
* **عزل المفاتيح السرية (API Security Key Guide):** تمر العمليات بممرات خوادم برمجية آمنة `/api/*` لحجب المفتاح السري بالكامل عن الواجهة البرمجية المباشرة للمستعرض.
* **الحماية من تكرار المعالجة الفجائي:** تم دمج مصفوفات التحقق المتكاملة والـ Error Catching في ممرات البث الفوري SSE لضمان استرجاع واجهات العميل لحالاتها الافتراضية بشكل تلقائي عند حدوث انقطاع غير متوقع للشبكة.
* **تحسين نفق الاستدعاءات وسياق الذكاء الاصطناعي:** يعتمد النظام على بروتوكول تحسين وسياق ثابت يمرر الحمض النووي (DNA) للمنشئ بصيغة منظمة للنموذج لتقليل استهلاك الـ Tokens وتسريع فترات الاستجابة لأقل من النصف.
